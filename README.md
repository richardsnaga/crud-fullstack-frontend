# Getting-Started

Clone this repo

### For database

First inject data with method POST and endpoint:
http://localhost:8080/user

#### For admin
```bash
{
    "username": "admin",
    "password": "admin",
    "email": "admin@gmail.com"
}
```

#### For user
```bash
{
    "username": "user1",
    "password": "user1",
    "email": "user1@gmail.com"
}
```

Then install the modules

```bash
npm install
```

Run the App

```bash
npm run start
```

## The application will start in
http://localhost:3000/
